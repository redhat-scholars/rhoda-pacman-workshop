Exploring Red Hat OpenShift Database Access

You can discover which database instance is connected to an application, along with other information by viewing the database inventory page.

Prerequisites

  • Already logged into the Red Hat Hybrid Cloud Console.

  • Access to the OpenShift Cluster Manager (OCM) console.

Procedure . Log into the Red Hat Hybrid Cloud Console with your credentials.

+ image::rhoda_console_login_screen.png[Red Hat Hybrid Cloud Console login page]

  1. Click OpenShift from the navigation menu.

  2. From the Clusters page, click on a cluster name from the list of available clusters.

  3. Click the Open console button, and provide your account credentials.

  4. From the OpenShift Cluster Manager console, change to the Administrator perspective.

  5. From the Project drop-down, select the redhat-dbaas-operator project.

  6. Expand the Data Services, and click on Database Access from the navigation menu.

  7. Viewing the inventory table you can find which database instance is connected to an application. You can also view the database provider, the provider account resource, and the associated project name.

    Figure 1. Example
